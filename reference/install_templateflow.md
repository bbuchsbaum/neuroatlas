# Install Templateflow (DEPRECATED)

\*\*DEPRECATED:\*\* TemplateFlow is now accessed via the pure R
\`templateflow\` package. Python is no longer required.

Install the R package instead:
`remotes::install_github("bbuchsbaum/templateflow")`

## Usage

``` r
install_templateflow(
  method = "auto",
  conda = "auto",
  envname = NULL,
  force_reinstall = FALSE
)
```

## Arguments

- method:

  Ignored.

- conda:

  Ignored.

- envname:

  Ignored.

- force_reinstall:

  Ignored.

## Value

Invisible `NULL`.
