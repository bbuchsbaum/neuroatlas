# Atlas Reference Accessor

Returns the canonical atlas reference metadata for an atlas object.

## Usage

``` r
atlas_ref(x, ...)

# S3 method for class 'atlas'
atlas_ref(x, ...)

# Default S3 method
atlas_ref(x, ...)
```

## Arguments

- x:

  An atlas object.

- ...:

  Additional arguments passed to methods.

## Value

An object of class \`"atlas_ref"\`.
