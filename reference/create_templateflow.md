# Create a TemplateFlow Interface Object

\*\*DEPRECATED:\*\* TemplateFlow is now accessed via the pure R
\`templateflow\` package. No initialization is needed. Use
\[get_template()\], \[tflow_spaces()\], and \[tflow_files()\] directly.

## Usage

``` r
create_templateflow(cache_dir = NULL, verbosity = 0, default_template = NULL)
```

## Arguments

- cache_dir:

  Ignored. Kept for backward compatibility.

- verbosity:

  Ignored. Kept for backward compatibility.

- default_template:

  Ignored. Kept for backward compatibility.

## Value

An S3 object of class `templateflow` (deprecated stub).

``` r
if (FALSE) { # \dontrun{
  # Deprecated. Use get_template(), tflow_spaces(), etc. directly.
} # }
```
