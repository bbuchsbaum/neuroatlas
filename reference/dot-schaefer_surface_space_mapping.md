# Map Schaefer surface space to CBIG and TemplateFlow identifiers

Map Schaefer surface space to CBIG and TemplateFlow identifiers

## Usage

``` r
.schaefer_surface_space_mapping(space)
```

## Arguments

- space:

  Character scalar, one of "fsaverage", "fsaverage5", "fsaverage6"

## Value

A named list with fields \`space\`, \`cbig_space\`, \`template_id\`,
\`tf_resolution\`, and \`tf_density\`.
