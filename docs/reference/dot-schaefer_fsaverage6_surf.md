# Schaefer surface atlas (fsaverage6 helper)

This internal helper preserves the existing behaviour of
[`get_schaefer_surfatlas()`](get_schaefer_surfatlas.md), using the
packaged `fsaverage` (fsaverage6) geometry plus Schaefer .annot label
files.

## Usage

``` r
.schaefer_fsaverage6_surf(parcels, networks, surf, use_cache = TRUE)
```
